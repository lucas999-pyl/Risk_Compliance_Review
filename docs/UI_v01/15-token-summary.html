Do

  • 所有"点我"信号 = Action Blue（CTA / link / focus）
  • 顶层节奏靠 white ↔ parchment ↔ dark 边到边交替分隔
  • 主 CTA = pill；工具按钮 = sm-8；卡片 = lg-18
  • display 尺寸（≥21px）加负字距 -0.224 → -0.5px
  • 状态用色点（dot 6px）+ 文字，不用色块填充
  • 报告页主结论卡是全系统唯一 drop-shadow

Don't

  • 引入第二品牌色（紫 / 绿 / 渐变 logo etc）
  • 红色按钮 · 红色背景填充（status-block 仅作色点 / 边）
  • emoji 类别图标 · 罗马数字 · 装饰渐变
  • 给卡片 / 按钮 / 文字加 box-shadow（仅主结论卡有）
  • 圆角中间值（4 / 12 / 16）· 字重 500
  • tile 加 border / shadow 当分隔（用色变即可）
